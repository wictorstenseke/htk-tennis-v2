### Audit and Enhance Tests for Critical Functionalities

The goal of this issue is to ensure that there is adequate test coverage for the following critical functionalities in the application:

1. Sign In 
2. Sign Up 
3. Sign Out 
4. Change/Edit Your Profile 
5. Book Match 
6. Challenge Players in Stegen 
7. Conflicts for Bookings if Time is Unavailable 
8. Admin Features

Some of these areas may already have existing tests. The following steps are suggested:

- Audit the current state of tests for each functionality.
- Identify missing test cases or potential edge case scenarios.
- Implement additional tests as needed to cover gaps.

### Acceptance Criteria
- [ ] All functionalities listed have been reviewed for test coverage.
- [ ] Missing test cases have been identified and implemented.

### Notes
- Ensure both unit tests and integration tests are considered where applicable.
- Please provide updates or findings in this issue thread.
